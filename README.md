# ⛩️ Samurai Scroll — Aravind Pyli's Portfolio

A cinematic, editorial portfolio website showcasing the intersection of **Artificial Intelligence**, **Computer Vision**, and **Product Engineering**. The design draws inspiration from minimalist Japanese aesthetics, featuring a custom scroll-driven canvas animation sequence, sleek micro-interactions, and detailed professional showcases.

---

## 🗡️ Features
- **Cinematic Scroll Sequence**: A responsive canvas-based image sequence (235 frames) showing a samurai stance and slash as you scroll down.
- **Dynamic Arsenal Ring**: A 3D-like circular carousel interface showcasing core skills and technologies.
- **Experience Scrolls**: Interactive scroll-driven display for professional experiences and fellowships.
- **Credentials & Badges**: Interactive certifications shelf highlighting credentials from Google, Harvard, LSE, and IBM.
- **Responsive Layout**: Designed for screens of all sizes, featuring adaptive layouts and a custom mobile drawer.
- **SEO Optimized**: Standard meta tags, descriptions, and semantic HTML built directly into routes.

---

## 🛠️ Tech Stack
- **Framework**: [TanStack Start](https://tanstack.com/router/v1/docs/start/overview) (Server-Side Routing & React Router v1)
- **Library**: React 19 & TypeScript
- **Styling**: Tailwind CSS v4 & Vanilla CSS variables
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: Lucide React
- **Package Manager / Runtime**: [Bun](https://bun.sh/) (with Vite)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Bun](https://bun.sh/) or [Node.js](https://nodejs.org/) installed.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aravindp1807/Protfolio.git
   cd Protfolio
   ```

2. Install dependencies:
   ```bash
   bun install
   # or npm install / yarn install
   ```

### Running Locally
Start the development server:
```bash
bun dev
# or npm run dev
```
Open [http://localhost:3000](http://localhost:3000) (or the port specified by Vite) in your browser.

### Building for Production
Build the optimized production app:
```bash
bun run build
# or npm run build
```

Preview the production build:
```bash
bun run preview
# or npm run preview
```
